# vueDemo
基于vue+vue-router+axios+element-ui的web端练习工程

## 运行（安装nodeJs6.10+）
```bash
git clone https://github.com/xiaoqiongxian/vueDemo.git
```
```bash
cd vueDemo
```
```bash
npm install
```
```bash
npm run dev
```
Navigate to [http://loaclhost:8080](http://loaclhost:8080) or [http://127.0.0.1:8080](http://127.0.0.1:8080)

## Build
```bash
npm run prod
```
